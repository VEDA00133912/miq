Make it a Quote API Documentation

エンドポイント

GET /

指定されたパラメータに基づいて画像を生成します。

リクエストURL: https://miq-dtsg.onrender.com/

リクエストパラメータ

パラメータ名 型 説明
type string 生成する画像のタイプを指定します
指定無しの場合はmonoが適用されます
カラーの場合→color モノクロの場合→mono
name string メッセージの送信者の名前を指定します
user_name string ユーザー名を指定します
content string 送信メッセージの内容を指定します
現在は長文の場合はみ出るので後日修正予定です
icon URL アイコン画像のURLを指定します

レスポンス例

image/pngで返されます

画像例を表示

エラーレスポンス

リクエストパラメータが無効な場合、400 Bad Requestとエラーメッセージが返されます。
