Make it a Quote API Documentation

エンドポイント

GET /

指定されたパラメータに基づいて画像を生成します。

リクエストURL: https://miq-dtsg.onrender.com/

リクエストパラメータ

パラメータ名 型 説明
type string 画像のタイプを指定します。「color」または「mono」
name string 引用者の名前を指定します
user_name string ユーザー名を指定します
content string 引用の内容を指定します
icon URL アイコン画像のURLを指定します

レスポンス例

画像例を表示

エラーレスポンス

リクエストパラメータが無効な場合、400 Bad Requestとエラーメッセージが返されます。
